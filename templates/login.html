Having Login Issues?

If you see "Access blocked: authorisation error", follow these steps:

  • Check Google Console: Verify OAuth credentials are set up correctly
  • Authorized Domains: Add your Render domain to Google Console
  • Redirect URI: Must match exactly: https://your-app.onrender.com/oauth2callback
  • Environment Variables: Ensure all Google OAuth variables are set in Render
